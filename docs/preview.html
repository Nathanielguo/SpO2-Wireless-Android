关于

这是一款通过 WiFi 无线传输的血氧双波监测软件。

操作方法：

1. 首先开启硬件 SpO2 监测设备，手机打开 WiFi 搜索 "WiFi_SpO2" 输入密码并连接；

2. 然后进入软件，点击【无线连接】输入正确的 IP 和端口号，连接 WiFi 通讯；

3. 返回首页，点击【开始监测】，即可进入页面显示实时血氧双波。
